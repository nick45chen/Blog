[Material Design] Android Toolbar(一)

Introduction

Toolbar 是 Google 在Android 5.0 推出的一個 Material Design 的控件，Toolbar 的靈活性在於並不只限於只能放在上方導航欄的位置上，同時也能嵌套在任意的 View 結構中，當然你可以在 Activity 中，呼叫 setActionBar() 方法，把 Toolbar 指定為上方導航欄使用。
